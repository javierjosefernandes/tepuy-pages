Política de Privacidad — Hatilú Studio

Tus datos se quedan en tu dispositivo y en tu cuenta personal de iCloud. Nunca vemos tus entrenamientos ni tu información personal.

Datos que recopilamos

Sincronización con iCloud

Tus entrenamientos, ejercicios, plantillas y ajustes se sincronizan entre tus dispositivos mediante iCloud (CloudKit) de Apple. Estos datos se almacenan en tu cuenta personal de iCloud y nunca son accesibles para nosotros.

Analítica de Uso

Recopilamos datos de uso anónimos para entender qué funciones importan más y mejorar la app para ti. Estos datos no pueden vincularse a tu identidad. Utilizamos Mixpanel para este fin.

Informes de Fallos

Si la app falla, se envía un informe automáticamente para que podamos encontrar y solucionar el problema. Estos informes no contienen información personal. Utilizamos Sentry para este fin.

Feature Flags

Utilizamos Flagsmith para gestionar la disponibilidad de funciones. No se recopilan datos personales a través de este servicio.

Datos que no recopilamos

  • No recopilamos tu nombre, correo electrónico ni ningún identificador personal
  • No vendemos ni compartimos tus datos con anunciantes
  • No utilizamos publicidad de terceros
  • No accedemos a datos de salud o fitness fuera de lo que introduces en la app

Servicios de terceros

Mixpanel — Analítica de uso anónima

Sentry — Informes de fallos

Flagsmith — Feature flags

Apple iCloud — Sincronización de datos

Tus derechos

Puedes eliminar todos tus datos en cualquier momento usando la opción «Borrar Todos los Datos» en Ajustes → Datos. Esto elimina tus datos tanto del dispositivo como de iCloud. También puedes gestionar tus datos de iCloud directamente desde los ajustes de iCloud de tu dispositivo.

Contacto

Si tienes preguntas sobre esta política de privacidad, contáctanos en tepuy@jfd.slmail.me.
