Política de Privacidade — Hatilú Studio

Os teus dados ficam no teu dispositivo e na tua conta pessoal do iCloud. Nunca vemos os teus treinos ou informações pessoais.

Dados que recolhemos

Sincronização iCloud

Os teus treinos, exercícios, modelos e definições são sincronizados entre os teus dispositivos através do iCloud (CloudKit) da Apple. Estes dados são armazenados na tua conta pessoal do iCloud e nunca são acessíveis para nós.

Estatísticas de Utilização

Recolhemos dados de utilização anónimos para perceber quais funcionalidades são mais importantes e melhorar a app para ti. Estes dados não podem ser associados à tua identidade. Utilizamos o Mixpanel para este efeito.

Relatórios de Erro

Se a app falhar, é enviado automaticamente um relatório para que possamos encontrar e corrigir o problema. Estes relatórios não contêm informações pessoais. Utilizamos o Sentry para este efeito.

Feature flags

Utilizamos o Flagsmith para gerir a disponibilidade de funcionalidades. Não são recolhidos dados pessoais através deste serviço.

Dados que não recolhemos

  • Não recolhemos o teu nome, endereço de e-mail ou quaisquer identificadores pessoais
  • Não vendemos nem partilhamos os teus dados com anunciantes
  • Não utilizamos publicidade de terceiros
  • Não acedemos a dados de saúde ou fitness fora do que introduzes na app

Serviços de terceiros

Mixpanel — Estatísticas de utilização anónimas

Sentry — Relatórios de erro

Flagsmith — Feature flags

Apple iCloud — Sincronização de dados

Os teus direitos

Podes eliminar todos os teus dados a qualquer momento utilizando a opção «Apagar Todos os Dados» em Definições → Dados. Isto remove os teus dados tanto do dispositivo como do iCloud. Também podes gerir os teus dados do iCloud diretamente através das definições de iCloud do teu dispositivo.

Contacto

Se tiveres questões sobre esta política de privacidade, contacta-nos em tepuy@jfd.slmail.me.
