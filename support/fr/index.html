Questions fréquentes

Comment fonctionne la synchronisation iCloud ?

Tepuy synchronise automatiquement tes entraînements, exercices, modèles et réglages sur tous tes appareils via ton compte iCloud personnel. Assure-toi qu’iCloud est activé pour Tepuy dans Réglages → iCloud.

Puis-je récupérer des données supprimées ?

Une fois les données supprimées via « Effacer toutes les données » dans Réglages → Données, elles sont définitivement supprimées de l’appareil et d’iCloud. Nous te recommandons d’être certain avant d’utiliser cette option.

Comment transférer mes données vers un nouvel appareil ?

Connecte-toi simplement avec le même compte Apple sur ton nouvel appareil et installe Tepuy. Tes données se synchroniseront automatiquement via iCloud.

L’app a planté ou quelque chose ne fonctionne pas

Écris-nous à tepuy@jfd.slmail.me avec une description du problème. Les rapports de crash sont envoyés automatiquement pour nous aider à diagnostiquer le problème.

Confidentialité

Tes données restent sur ton appareil et dans ton compte iCloud personnel. Consulte notre Politique de confidentialité complète.
