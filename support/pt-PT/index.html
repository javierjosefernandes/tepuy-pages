Perguntas frequentes

Como funciona a sincronização com iCloud?

O Tepuy sincroniza automaticamente os teus treinos, exercícios, modelos e definições em todos os teus dispositivos através da tua conta pessoal de iCloud. Certifica-te de que o iCloud está ativado para o Tepuy em Definições → iCloud.

Posso recuperar dados eliminados?

Depois de eliminados os dados através de “Apagar Todos os Dados” em Definições → Dados, estes são removidos permanentemente do dispositivo e do iCloud. Recomendamos que tenhas a certeza antes de utilizar esta opção.

Como transfiro os meus dados para um novo dispositivo?

Basta iniciares sessão com a mesma conta Apple no teu novo dispositivo e instalar o Tepuy. Os teus dados serão sincronizados automaticamente através do iCloud.

A app fechou ou algo não está a funcionar

Por favor escreve-nos para tepuy@jfd.slmail.me com uma descrição do que aconteceu. Os relatórios de crash são enviados automaticamente para nos ajudar a diagnosticar o problema.

Privacidade

Os teus dados ficam no teu dispositivo e na tua conta pessoal de iCloud. Lê a nossa Política de Privacidade completa.
